## English

### Overview
A free, open-source tool designed to verify DOI citations generated by AI tools (ChatGPT, Claude, Perplexity) and convert them into standard bibliographic formats (RIS/BibTeX/CSV) compatible with EndNote and Zotero.

### Problem Statement
Large Language Models occasionally generate fictitious citations that appear legitimate but do not correspond to actual published works—a phenomenon known as "AI hallucination." Inadvertent use of such citations can compromise research integrity and academic credibility.

### Solution
This tool addresses the issue through:
- **Verification**: Validates each DOI against the CrossRef database in real-time
- **Conversion**: Exports verified citations to RIS, BibTeX, or CSV formats
- **Integration**: Enables direct import into reference management software (EndNote, Zotero)

## 中文

### 概述
免費開源工具，專門用於驗證 AI 工具（ChatGPT、Claude、Perplexity）產生的 DOI 文獻真實性，並將其轉換為標準書目格式（RIS/BibTeX/CSV），可直接匯入 EndNote 和 Zotero。

### 問題背景
大型語言模型有時會產生看似合理但實際上並不存在的虛構文獻引用，此現象稱為「AI 幻覺」。若不慎引用此類文獻，可能損害研究品質與學術信譽。

### 解決方案
本工具透過以下機制解決此問題：
- **驗證功能**：即時透過 CrossRef 資料庫驗證每筆 DOI 的真實性
- **格式轉換**：將驗證通過的文獻轉換為 RIS、BibTeX 或 CSV 格式
- **無縫整合**：可直接匯入文獻管理軟體（EndNote、Zotero）

## Technical Specifications

### Architecture
- **Frontend**: Vanilla JavaScript with inline CSS
- **API**: CrossRef REST API (public, no authentication required)
- **Deployment**: GitHub Pages (static hosting)
- **Dependencies**: None (zero external libraries)

### Features
- Real-time DOI validation
- Batch processing support
- Multiple export formats (RIS, BibTeX, CSV)
- Client-side processing (privacy-preserving)
- CrossRef API compliance
