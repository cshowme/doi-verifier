## English

### Overview
A free tool designed to verify DOI citations generated by AI tools (ChatGPT, Claude, Perplexity) and convert them into standard bibliographic formats. Prevent "AI hallucination" citations from compromising your research.

### What's New in v7.6

| Feature | Description |
|---------|-------------|
| **Smart DOI Cleaning** | Auto-normalize various formats (doi:, doi.org/, DOI:, trailing punctuation) |
| **Duplicate Detection** | Automatically merge duplicate DOIs |
| **Citation Formats** | Generate APA, Vancouver, MLA citations with one click |
| **Batch Copy** | Copy all citations at once |
| **Free Plan System** | 30 DOIs per run, 3 runs per 15-minute window |

### Problem Statement
Large Language Models occasionally generate fictitious citations that appear legitimate but do not correspond to actual published works—a phenomenon known as "AI hallucination." Inadvertent use of such citations can compromise research integrity and academic credibility.

### Solution
This tool addresses the issue through:
- **Verification**: Validates each DOI against the CrossRef database (150M+ records) in real-time
- **Smart Cleaning**: Auto-fix common DOI format issues
- **Deduplication**: Detect and merge duplicate entries
- **Citation Generation**: APA, Vancouver, MLA formats
- **Conversion**: Exports verified citations to RIS, BibTeX, or CSV formats
- **Integration**: Enables direct import into reference management software (EndNote, Zotero, Mendeley)

### Access
**Live Tool**: https://cshowme.github.io/doi-verifier/

### Quick Start
1. Ask AI for literature with DOIs: *"Please provide DOIs in format 10.xxxx/xxxxx"*
2. Paste DOI list into the tool
3. Click "開始驗證" (Start Verification)
4. Download RIS/BibTeX or copy citations

## Technical Specifications

### Architecture
- **Frontend**: Vanilla JavaScript with inline CSS (single-file deployment)
- **API**: CrossRef REST API (public, polite pool with mailto)
- **Deployment**: GitHub Pages (static hosting)
- **Dependencies**: None (zero external libraries)

### Features
| Feature | Description |
|---------|-------------|
| Real-time DOI validation | CrossRef API integration |
| Smart DOI parsing | Regex-based extraction with auto-cleanup |
| Duplicate detection | Set-based deduplication |
| Citation formatting | APA, Vancouver, MLA support |
| Multiple export formats | RIS, BibTeX, CSV |
| Client-side processing | Privacy-preserving, no data storage |
| Free plan management | localStorage-based rate limiting |

### Browser Compatibility
Modern browsers (Chrome, Firefox, Safari, Edge)

### API Information
- **Provider**: CrossRef (https://www.crossref.org/)
- **Endpoint**: `https://api.crossref.org/works/{doi}`
- **Rate Limit**: Polite pool (300ms delay between requests)
